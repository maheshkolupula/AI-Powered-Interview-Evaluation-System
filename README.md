# AI-Powered Interview Evaluation System

An AI-based smart interview simulator that automatically conducts online interviews by asking questions, recording candidate responses, converting speech to text, analyzing facial emotions, and generating real-time performance feedback using Python, AI, and Machine Learning.

# Purpose

This application is designed to reduce the burden on employers by automating the initial interview and evaluation process. It helps organizations quickly screen candidates while providing applicants with a realistic interview practice environment.
